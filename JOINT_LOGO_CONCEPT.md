# JOINT LOGO CONCEPT: Over-Simplify × Mentoria

---

## Logo Design Direction

### Option 1: Dual-Line Lockup (Recommended)

```
┌────────────────────────────────────┐
│                                    │
│      OVER-SIMPLIFY CAREERS         │
│            with MENTORIA           │
│                                    │
└────────────────────────────────────┘
```

**Typography:**
- "OVER-SIMPLIFY CAREERS" — Bold, modern sans-serif (e.g., Montserrat Bold, Inter Bold)
- "with MENTORIA" — Lighter weight or slightly smaller size to show partnership hierarchy
- Color: Primary brand color for Over-Simplify, Mentoria's brand color for "MENTORIA"

**Layout:**
- Centered alignment
- Clean, professional spacing
- Works well on letterheads, websites, and presentations

---

### Option 2: Side-by-Side Lockup

```
┌────────────────────────────────────┐
│                                    │
│  OVER-SIMPLIFY  ×  MENTORIA        │
│      CAREERS                       │
│                                    │
└────────────────────────────────────┘
```

**Typography:**
- Equal visual weight between both brands
- "×" or "&" as separator to show collaboration
- "CAREERS" as descriptor below

**Use Case:** For horizontal layouts (website headers, email signatures)

---

### Option 3: Icon + Text Combination

```
┌────────────────────────────────────┐
│        🎓                          │
│   OVER-SIMPLIFY CAREERS            │
│   Powered by MENTORIA              │
│                                    │
└────────────────────────────────────┘
```

**Typography:**
- Icon/graphic element representing education/guidance
- "Powered by MENTORIA" shows Mentoria as the technology/assessment provider
- Over-Simplify as the delivery/implementation partner

---

## Color Palette Suggestions

### Over-Simplify Colors:
- **Primary:** #667eea (Purple-Blue — represents clarity, trust, innovation)
- **Secondary:** #764ba2 (Deep Purple — represents depth and expertise)
- **Accent:** #00d4ff (Bright Blue — represents accessibility and modernity)

### Mentoria Colors:
- Use Mentoria's official brand colors for consistency
- If unknown, suggest neutral professional colors like #2c3e50 (Navy) or #27ae60 (Green for growth)

---

## Typography Recommendations

**Primary Font:** Montserrat, Inter, or Poppins  
- Modern, clean, highly legible
- Works well across digital and print

**Secondary Font:** Open Sans or Roboto  
- For body text and supporting content

---

## Logo Usage Guidelines

### Primary Use:
- Website header (oversimplify.in)
- Email signatures
- Presentations to schools
- Marketing collateral (brochures, flyers)

### Minimum Size:
- Print: 1 inch wide
- Digital: 200px wide
- Ensure legibility at all sizes

### Clear Space:
- Maintain minimum spacing around logo equal to the height of the "M" in MENTORIA

### Background Use:
- Logo should work on white, light gray, and dark backgrounds
- Provide reverse (white) version for dark backgrounds

   - SVG (vector, scalable)

   - JPG (white background, high-res)
   - Favicon format (for website)

---

## Quick DIY Option (If No Designer Available)

Use a tool like Canva:
1. Create a 1200x600px canvas
2. Use Montserrat Bold font
3. Type: "OVER-SIMPLIFY CAREERS" in #667eea
4. Below, type: "with MENTORIA" in #27ae60 (or Mentoria's brand color)
5. Export as PNG with transparent background

This provides a clean, professional interim logo for the approval email and initial marketing materials.
